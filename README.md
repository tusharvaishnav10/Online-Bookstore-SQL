# 📚 Online Bookstore SQL Project

## 📌 About
SQL project analyzing an Online Bookstore database 
with Books, Customers, and Orders data.

## 🗃️ Tables
- Books
- Customers
- Orders

## 📊 What I Analyzed
- Total revenue from all orders
- Top customers by spending
- Most popular books
- Books sold by genre and author
- Stock remaining after orders
- Customers with multiple orders

## 🛠️ Tools Used
- PostgreSQL
- SQL (Joins, Group By, Having, Coalesce)
